Команды

docker info [OPTIONS] вывести информацию о докере и хосте
docker version [OPTIONS] версия докера

docker ps [OPTIONS] информация по текущим запущенным контейнерам, списко контейнеров
docker ps -a все контейнеры
 ID 	    ID контейнера
 Image 	    ID образа
 Command    Команда
 Created 	Время когда контенер был создан
 Ports 	    Открытые порты
 Status 	Статус контейнера
 Names 	    Имя контенера

docker images [OPTIONS] [REPOSITORY[:TAG]] список локальных образов

docker search [OPTIONS] TERM поиск образа

docker pull [OPTIONS] NAME[:TAG|@DIGEST] скачать образ из реестра
docker pull mysql:5.7 скачаем определенную версию mysql из репозитория

docker run [OPTIONS] IMAGE [COMMAND] [ARG...] Cамая важная и сложная команда
docker run -t -i ubuntu /bin/bash при таком приеме контейнер останавливается
docker run -d ubuntu /bin/sh запуск контейнера как демона
-a
-d Выполняет контейнер в "отключённом режиме". Команда будет выполнять контейнер в фоновом режиме и вернёт идентификатор контейнера.
-i интерактивный запуск контейнера
--rm удаляет контейнер после запуска
-t - вывести консоль
-e - переменные окружения для контейнера
-h - имя хоста контейнера
--name назначить контейнеру имя
-p - публикует порт контейнера

При запуск докер будет искать образ если ее нет то он будет скачан

docker logs Container что происходит внутри контейнера
